# Sistema de Controle de Vendas e de Estoque
## Sobre o Projeto

O Sistema de Controle de Vendas e de Estoque foi desenvolvido como parte da 5ª Etapa do Trabalho Interdisciplinar: Aplicações para Processos de Negócios (TIAPN) do curso de Sistemas de Informação da PUC Minas – Unidade Betim.

O projeto tem como objetivo auxiliar um microempreendimento de revenda de queijos no gerenciamento de produtos, clientes, fornecedores, estoque e vendas por meio de uma aplicação centralizada.

## Problema

Antes da implementação do sistema, o controle das operações era realizado por meio de anotações e registros manuais, tornando difícil o acompanhamento das informações do negócio.

As principais dificuldades identificadas foram:

Informações dispersas;
Controle fragmentado de produtos, clientes e fornecedores;
Dificuldade no acompanhamento do estoque;
Maior risco de perda de informações;
Lentidão na tomada de decisões.

## Objetivos

O sistema foi desenvolvido para:

Centralizar as informações do negócio;
Facilitar o cadastro de produtos, clientes e fornecedores;
Controlar entradas e saídas de estoque;
Registrar vendas;
Melhorar a rastreabilidade das movimentações;
Apoiar a gestão do microempreendimento.

## Funcionalidades
Login de usuários;
Cadastro de produtos;
Cadastro de clientes;
Cadastro de fornecedores;
Dashboard com indicadores;
Registro de entradas de estoque;
Registro de vendas;
Atualização automática do estoque;
Histórico de entradas;
Histórico de vendas.

## Funcionamento

Os módulos do sistema são integrados, permitindo que cada movimentação realizada atualize automaticamente as informações do estoque e mantenha o histórico das operações.

O dashboard apresenta uma visão resumida das principais informações do negócio, facilitando o acompanhamento diário e a tomada de decisões.
